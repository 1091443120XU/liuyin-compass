# 留音 Compass

一个面向计划留学的音乐艺术生的信息整合与院校推荐原型。

## 功能

- 录入申请阶段、主修方向、目标地区、预算、语言成绩、GPA、Gap 与奖学金偏好
- 根据院校报考要求、学费、语言门槛、Gap 友好度、专业覆盖和排名做匹配评分
- 按目标地区和学校类型进行硬筛选，支持区分音乐学院/专门艺术院校与综合类大学音乐学院
- 查看院校详情：招生条例、考学内容、语言要求、课程安排、学制、位置、排名与来源
- 在院校详情中查看各专业的官网入口，直接跳转学校课程页或招生页复核
- 加入对比清单，快速比较学费、语言、排名和适配点

## 运行

直接用浏览器打开 `index.html` 即可运行。

## 数据维护

院校种子数据在 `app.js` 的 `schools` 数组中。每所学校都保留 `sources` 字段，后续可以替换为后端数据库、CMS、表格导入或定时抓取审核流程。

当前数据版本：2026-06-05，已扩展到 QS Music 2026 榜单院校、常见申请目标校与更多地区代表性院校。学费、语言成绩、曲目、专业链接和截止日期会频繁变化，正式用于申请规划前需要以院校官网最新信息复核。
